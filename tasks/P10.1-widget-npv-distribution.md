# Task P10.1: NPV distribution histogram widget

## Objective

Build an interactive histogram widget showing the full distribution of individual NPV outcomes from the synthetic population. This is the signature Phase 2 widget — it transforms the report from showing single NPV point estimates to revealing the complete range of fiscal outcomes, enabling probability statements like "72% of skilled migrants arriving at 30 are net contributors exceeding $100k."

## Inputs

- `data/output/synth-npv-distributions.json` — histogram bin data, percentiles, and summary statistics per visa type and arrival age
- `client/src/components/charts/NPVCalculatorWidget.tsx` — reference for the existing widget pattern (useQuery, SVG charting, Tailwind styling, named export)
- `client/src/components/charts/chartRegistry.ts` — reference for lazy import pattern (do NOT modify; P10.5 handles registration)

## Expected outputs

- `client/src/components/charts/NPVDistributionWidget.tsx` — a single self-contained React component

### Step 3: Build the component

Create `client/src/components/charts/NPVDistributionWidget.tsx` with:

1. **Imports**: React, useState, useQuery from @tanstack/react-query, cn from @/lib/utils

2. **Data fetching**: useQuery for `/data/synth-npv-distributions.json`

3. **Controls**:
   - Visa type selector: `<select>` dropdown with all visa types
   - Arrival age selector: `<select>` dropdown (20, 25, 30, ... 55)
   - NZ-born overlay toggle: checkbox or toggle button

4. **Histogram chart** (SVG, no D3):
   - 50 bins spanning the NPV range
   - SVG bars colored teal (#6EA1BE) for migrant distribution
   - When NZ-born overlay is enabled, overlay NZ-born histogram with steel (#6F899A) bars at 50% opacity, dashed outline
   - Vertical reference line at NZ-born mean, labeled, styled steel dashed
   - X-axis: NPV in NZD (formatted with $, K/M suffixes)
   - Y-axis: frequency count or density
   - Responsive SVG with viewBox

5. **Callout text**:
   - Large text: "X% are net contributors exceeding $100k" — dynamically from data
   - Color: sage (#7A9E7E) if >50%, coral (#F26B44) if <50%

6. **Percentile stat cards**: Row of 5 cards showing p10, p25, median, p75, p90
   - Each card: label on top (steel text), value below (navy text, bold)
   - Cards have subtle border, rounded corners
   - Negative values in coral, positive in sage

7. **Loading state**: Skeleton with animate-pulse matching the layout
8. **Error state**: Styled error message
