# Task P1.1: Extract Hughes Table 1 (aggregate tax by transcat/age/year)

## Objective
Parse the Hughes AN 26/02 Excel file and extract Table 1 into a clean JSON file. Table 1 contains aggregate population counts and tax paid, broken down by year, 20-year age bands, and transnational category (e.g., Resident Birth Citizen, Resident Temporary Worker, Diasporic Birth Citizen, etc.).

## Inputs
- `data/raw/hughes-an2602.xlsx` — Sheet "Data", rows where `table_nbr == 1`

## Expected outputs
- `data/processed/hughes-table1-aggregate.json`

## Execution instructions

Write a Python script `analysis/01-extract-hughes-table1.py` that:

1. Opens `data/raw/hughes-an2602.xlsx` using openpyxl (read_only=True for speed — it's 19MB).
2. Reads the "Data" sheet. Column structure (row 1 is headers):
   - A: table_nbr
   - B: var1 (= "yr" for Table 1)
   - C: var1_value (= year, integer)
   - D: var2 (= "NULL" for Table 1)
   - E: var2_value
   - F: var3 (= "age_20bins" for Table 1)
   - G: var3_value (= age band start: 0, 20, 40, 60, 80)
   - H: var4 (= "transcat")
   - I: var4_value (= transnational category name, e.g. "Resident Birth Citizen")
   - J: measure_type1 (= "underlying count")
   - K: measure1 (= count of people, integer, or "S" if suppressed)
   - L: measure_type2 (= "tax_b")
   - M: measure2 (= tax paid in billions of dollars, float, or 0)
3. Filter to rows where table_nbr == 1.
4. Skip rows where measure1 == "S" (suppressed by Stats NZ).
5. Build a list of objects:
   ```json
   {
     "year": 2024,
     "age_band": "20-39",
     "age_start": 20,
     "transcat": "Resident Birth Citizen",
     "count": 844899,
     "tax_billions": 4.44525
   }
   ```
   Map age_start values: 0→"0-19", 20→"20-39", 40→"40-59", 60→"60-79", 80→"80+"
6. Write to `data/processed/hughes-table1-aggregate.json` with indent=2.
7. Print summary: total rows extracted, year range, distinct transcats, total tax in latest year.

Then run the script: `python3 analysis/01-extract-hughes-table1.py`

## Self-check
- JSON file should load successfully and contain ~2,500-2,700 records (raw table has ~2,698 rows minus suppressed)
- Year range should be 1999-2025
- Distinct transcats should include: Resident Birth Citizen, Resident Resident, Resident Temporary Worker, Diasporic Birth Citizen, Former Resident, Visitor, etc.
- Tax for "Resident Birth Citizen" age 20-39 in year 2024 should be approximately 4.45 billion

## Handoff
- `data/processed/hughes-table1-aggregate.json` is consumed by P1.7 (matching/NPV) and P2.3 (findings writing)
