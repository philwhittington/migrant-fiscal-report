# Task P1.4: Extract Hughes Tables 14 and 142 (retention/survival curves)

## Objective
Extract cohort tracking data that shows how many migrants from each arrival cohort are still resident in NZ in subsequent years. This data powers the retention curves — the probability of a migrant still being in NZ at year+1, +2, +5, +10, etc. — which is a critical input to the NPV lifecycle model.

## Inputs
- `data/raw/hughes-an2602.xlsx` — Sheet "Data", rows where table_nbr in (14, 142)

## Expected outputs
- `data/processed/hughes-table14-cohort-visa.json` — Raw cohort tracking by first visa type
- `data/processed/hughes-table142-cohort-age.json` — Raw cohort tracking by arrival age
- `data/processed/retention-curves-by-visa.json` — Derived retention rates by visa type
- `data/processed/retention-curves-by-age.json` — Derived retention rates by arrival age

## Execution instructions

Write a Python script `analysis/04-extract-hughes-retention.py`:

### Part A: Extract raw tables

**Table 14** (~69,000 rows):
- var1=yr, var1_value=observation year (int)
- var2=arrival_year, var2_value=year of first migration (int)
- var3=transcat, var3_value=transnational category in observation year
- var4=first visa type, var4_value=visa at time of first migration
- measure1=count (skip if "S"), measure2=NULL (no tax in this table)
- Output: `{obs_year, arrival_year, transcat, first_visa, count}`

**Table 142** (~65,000 rows):
- var1=yr, var1_value=observation year (int)
- var2=arrival_year, var2_value=year of first migration (int)
- var3=transcat, var3_value=transnational category
- var4=arrival_age_10bins, var4_value=age at arrival (0,10,20,...) (int)
- measure1=count (skip if "S")
- Output: `{obs_year, arrival_year, transcat, arrival_age_start, count}`

### Part B: Compute retention curves

From Table 14, build retention curves by first visa type:

1. For each first_visa and arrival_year, compute the initial cohort size: sum of count where obs_year == arrival_year.
2. For each subsequent obs_year, compute the number still resident: sum of count where transcat starts with "Resident" (any of: Resident Australian, Resident Birth Citizen, Resident Non-birth Citizen, Resident Permanent Resident, Resident Resident, Resident Student etc, Resident Temporary Worker, Resident Unknown).
3. Retention rate = still_resident / initial_cohort.
4. For each first_visa, average the retention rates across cohorts 2000-2019 (to get stable estimates with at least 5 years of follow-up).
5. Output structure:
   ```json
   {
     "first_visa": "R.Skilled/investor/entrepreneu",
     "years_since_arrival": 5,
     "retention_rate": 0.84,
     "cohorts_averaged": 20,
     "initial_cohort_avg": 7500
   }
   ```

Do the same from Table 142 for retention by arrival age band.

**Key visa types to include**: R.Skilled/investor/entrepreneu, R.Family, R.Humanitarian and Pacific, W.Skills/specific purposes/pos, W.Working holiday, S.Fee paying, S.Dependent, A.Australian, C.Non_birth_citizen, V.Visitor, W.Family

### Part C: Fit extrapolation parameters

For each visa type, fit an exponential decay to the retention curve (years 10-18) to enable extrapolation beyond 18 years. Use: `retention(t) = a * exp(-b * t)`. Store the fitted parameters a, b for each visa type in the output.

Run the script.
