# Task P1.2: Extract Hughes Tables 4, 5, 7 (tax distributions by visa/age/sex)

## Objective
Extract three related tables from the Hughes dataset that contain tax payment information broken down by visa type, age, sex, and quantile. These tables are the primary source for understanding how much tax different migrant groups pay.

## Inputs
- `data/raw/hughes-an2602.xlsx` — Sheet "Data", rows where table_nbr in (4, 5, 7)

## Expected outputs
- `data/processed/hughes-table4-visa-subcategory.json`
- `data/processed/hughes-table5-visa-quantiles.json`
- `data/processed/hughes-table7-sex-visa-quantiles.json`

## Execution instructions

Write a Python script `analysis/02-extract-hughes-visa-tax.py` that extracts all three tables in one pass through the Excel file.

### Table 4: Tax by year, age, visa subcategory
- Filter: table_nbr == 4
- var1=yr, var1_value=year (int)
- var2=age_10bins, var2_value=age band start (0,10,20,...,90) (int)
- var3=visa_subcategory, var3_value=detailed visa (e.g. "R.Skilled/investor/entrepreneu", "W.Working holiday", "S.Fee paying")
- var4=visa_category, var4_value=broad visa (e.g. "Resident", "Non-residential work", "Student")
- measure1=count (skip if "S"), measure2=tax_billions (float)
- Output structure: `{year, age_start, visa_subcategory, visa_category, count, tax_billions}`
- Expected ~5,145 rows

### Table 5: Tax quantiles by visa category
- Filter: table_nbr == 5
- var1=taxyr, var1_value=tax year (int)
- var2=age_10_bins, var2_value=age band start (int)
- var3=visa_category, var3_value=broad visa category (10 categories: Australian, Birth Citizen, Diplomatic etc, Non-birth Citizen, Non-residential work, Permanent Resident, Resident, Student, Unknown (Presumed resident), Visitor)
- var4=quantile, var4_value=quantile name (p10, p25, p50, p75, p90)
- measure1=underlying count (skip if "S"), measure2=tax at that quantile in dollars (NOT billions — already in dollars for quantile tables)
- Output structure: `{taxyr, age_start, visa_category, quantile, count, tax_dollars}`
- Expected ~11,100 rows

### Table 7: Tax quantiles by sex and detailed visa
- Filter: table_nbr == 7
- var1=sex, var1_value="Male" or "Female"
- var2=age_10_bins, var2_value=age band start (int)
- var3=visa_subcategory, var3_value=detailed visa (29 subcategories)
- var4=quantile, var4_value=quantile name
- measure1=count (skip if "S"), measure2=tax in dollars
- Output structure: `{sex, age_start, visa_subcategory, quantile, count, tax_dollars}`
- Expected ~1,880 rows

Run the script after writing it.
