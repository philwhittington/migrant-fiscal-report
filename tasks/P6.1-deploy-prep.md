### Step 2: Create serve.sh

Create a `serve.sh` script at the project root that serves the built site locally for preview.

First determine the correct directory to serve. Check the build output:
- If `dist/public/` contains `index.html`, serve `dist/public`
- If `dist/` contains `index.html`, serve `dist`
- If `dist/client/` contains `index.html`, serve `dist/client`

Write `serve.sh`:
```bash
#!/bin/bash
# Serve the built migrant fiscal report locally for preview
# Usage: ./serve.sh
# Then open http://localhost:3000

set -e

SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
BUILD_DIR="$SCRIPT_DIR/dist/public"  # adjust based on actual build output

if [ ! -d "$BUILD_DIR" ]; then
  echo "Build directory not found. Run 'npm run build' first."
  exit 1
fi

echo "Serving migrant fiscal report from $BUILD_DIR"
echo "Open http://localhost:3000 in your browser"
echo ""

npx serve "$BUILD_DIR" -l 3000
```

Make it executable:
```bash
chmod +x serve.sh
```

Test that it starts without errors (briefly, then stop it):
```bash
cd /Users/PhilWhittington/Projects/migrant-fiscal-report
timeout 5 ./serve.sh || true
```
