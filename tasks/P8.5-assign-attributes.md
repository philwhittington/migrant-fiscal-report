# Task P8.5: Assign attributes

## Objective
Assign relationship type, nationality, and years-since-residence to every individual in the seed population. These attributes are needed for fiscal incidence (P8.6): relationship determines family linkage, nationality feeds convergence analysis, and tenure drives NZ Super eligibility and benefit stand-down rules.

## Inputs
- `synth-pop/seed-population.parquet` (P8.4 output) — columns: `id`, `age_start`, `visa_subcategory`, `visa_category`, `gross_income`, `income_tax`, `acc_levy`, `net_income`
- `synth-pop/assignment-tables.json` (P8.3 output) — contains:
  - `nationality_marginal`: P(nationality) unconditional
  - `relationship_given_nationality`: P(relationship | nationality)
  - `tenure_distribution`: P(tenure | relationship, age_start)
- `synth-pop/config.py` — import constants including `TEMP_VISA_CATEGORIES`

## Expected outputs
- `synth-pop/seed-population.parquet` — **updated in place** with new columns:
  - `relationship` (str): one of "Self", "Presumed Spouse", "Presumed Child"
  - `nationality` (str): one of the 11 nationality groups
  - `years_since_residence` (int): years since first NZ residence (drawn from tenure distribution)

## Execution instructions

### Step 1: Load inputs
```python
import json
import numpy as np
import pandas as pd
from synth_pop.config import TEMP_VISA_CATEGORIES

seed = pd.read_parquet('synth-pop/seed-population.parquet')

with open('synth-pop/assignment-tables.json') as f:
    tables = json.load(f)

nat_marginal = tables['nationality_marginal']
rel_given_nat = tables['relationship_given_nationality']
tenure_dist = tables['tenure_distribution']
```

### Step 2: Assign nationality
Draw nationality for each individual from the marginal distribution. This is unconditional — Table 10 does not break down by visa category, so we use the aggregate distribution.

```python
rng = np.random.default_rng(seed=43)

nationalities = list(nat_marginal.keys())
nat_probs = np.array([nat_marginal[n] for n in nationalities])
nat_probs /= nat_probs.sum()  # ensure normalised

seed['nationality'] = rng.choice(nationalities, size=len(seed), p=nat_probs)
```

### Step 3: Assign relationship
For each individual, draw relationship conditional on their nationality. However, also apply age constraints:
- Individuals with `age_start` in [0, 10] are forced to "Presumed Child" regardless of draw.
- Individuals with `age_start` >= 60 are forced to "Self" (spouse assignment is rare and adds noise at old ages).

```python
def draw_relationship(nationality, age_start):
    # Age-based override
    if age_start <= 10:
        return 'Presumed Child'
    if age_start >= 60:
        return 'Self'

    probs_dict = rel_given_nat.get(nationality, {'Self': 0.5, 'Presumed Spouse': 0.3, 'Presumed Child': 0.2})
    rels = list(probs_dict.keys())
    probs = np.array([probs_dict[r] for r in rels])
    probs /= probs.sum()
    return rng.choice(rels, p=probs)

seed['relationship'] = [
    draw_relationship(row['nationality'], row['age_start'])
    for _, row in seed.iterrows()
]
```

**Performance note**: The row-wise loop above is slow on 500k rows. Consider vectorising by pre-computing the draws per (nationality, age_start) group:

```python
seed['relationship'] = ''
for nat in nationalities:
    for age in seed['age_start'].unique():
        mask = (seed['nationality'] == nat) & (seed['age_start'] == age)
        n = mask.sum()
        if n == 0:
            continue
        if age <= 10:
            seed.loc[mask, 'relationship'] = 'Presumed Child'
        elif age >= 60:
            seed.loc[mask, 'relationship'] = 'Self'
        else:
            probs_dict = rel_given_nat.get(nat, {'Self': 0.5, 'Presumed Spouse': 0.3, 'Presumed Child': 0.2})
            rels = list(probs_dict.keys())
            probs = np.array([probs_dict[r] for r in rels])
            probs /= probs.sum()
            seed.loc[mask, 'relationship'] = rng.choice(rels, size=n, p=probs)
```

### Step 4: Assign years since residence
Draw from the tenure distribution conditioned on (relationship, age_start). The tenure bands in Table 11 are 0, 5, 10, 15, 20. Draw the band, then assign a uniform random year within that band.

```python
seed['years_since_residence'] = 0

for rel in ['Self', 'Presumed Spouse', 'Presumed Child']:
    for age in seed['age_start'].unique():
        mask = (seed['relationship'] == rel) & (seed['age_start'] == age)
        n = mask.sum()
        if n == 0:
            continue

        key = f"{rel}|{age}"
        if key not in tenure_dist:
            # Fallback: use closest available age
            available_ages = [int(k.split('|')[1]) for k in tenure_dist if k.startswith(f"{rel}|")]
            if available_ages:
                closest = min(available_ages, key=lambda a: abs(a - age))
                key = f"{rel}|{closest}"
            else:
                seed.loc[mask, 'years_since_residence'] = 0
                continue

        tenures = tenure_dist[key]
        bands = list(tenures.keys())
        probs = np.array([tenures[b] for b in bands])
        probs /= probs.sum()

        # Draw band
        drawn_bands = rng.choice([int(b) for b in bands], size=n, p=probs)
        # Add uniform jitter within 5-year band
        jitter = rng.integers(0, 5, size=n)
        seed.loc[mask, 'years_since_residence'] = drawn_bands + jitter
```
