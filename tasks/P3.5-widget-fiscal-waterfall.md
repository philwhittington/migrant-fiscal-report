# Task P3.5: Fiscal incidence waterfall widget

## Objective

Build a Fiscal Incidence Waterfall widget that decomposes the fiscal impact of a migrant into its component parts: starting from market income, through taxes paid and government spending received, to arrive at the net fiscal impact. This is a classic waterfall chart that makes the fiscal story intuitive — viewers can see exactly where migrants contribute (taxes) and where they draw (health, education, NZ Super).

## Inputs

- `data/output/fiscal-components-by-migrant-type.json` — fiscal components (income, taxes, spending categories) for each migrant type at different arrival ages
- `client/src/components/charts/VATDemoWidget.tsx` — reference for existing widget pattern
- `client/src/components/charts/chartRegistry.ts` — reference for registration (do NOT modify; P3.6 handles this)

## Expected outputs

- `client/src/components/charts/FiscalWaterfallWidget.tsx` — the React widget component

## Execution instructions

If the file does not exist, create a realistic placeholder. Use plausible NZ fiscal values: average income ~$75k for skilled age 30, lower for humanitarian; income tax ~25% of income; GST ~8% effective rate; health ~$4k per person; education varies by age.

### Step 2: Read the existing widget pattern

Read `client/src/components/charts/VATDemoWidget.tsx` for data fetching, loading/error, Tailwind styling, export patterns.

### Step 3: Build the component

Create `client/src/components/charts/FiscalWaterfallWidget.tsx` with:

1. **Imports**: React, useState, useMemo, useQuery from @tanstack/react-query, cn from @/lib/utils

2. **Data fetching**: useQuery to fetch `/data/fiscal-components-by-migrant-type.json`

3. **Controls**:
   - Migrant type selector: dropdown with all available migrant type labels (e.g., "Skilled, age 30", "Family, age 30", "NZ-born, age 30", etc.). Default to "Skilled, age 30".
   - Toggle: "Show NZ-born comparison" checkbox (checked by default). When checked, shows a second waterfall alongside for "NZ-born, age 30" (matched to the selected arrival age if available).

4. **Waterfall chart** (plain SVG, no external library):

   The waterfall works as follows:
   - Start bar: "Market income" — full positive bar from zero
   - Step bars: each subsequent component either adds or subtracts from the running total
     - Income tax: subtracts (bar drops, colored coral)
     - Indirect tax: subtracts (bar drops, colored coral)
     - Income support: adds (bar rises, colored sage)
     - Education: adds (bar rises, colored sage)
     - Health: adds (bar rises, colored sage)
     - NZ Super: adds (bar rises, colored sage)
     - Other gov spending: adds (bar rises, colored sage)
   - Final bar: "Net fiscal impact" — total bar from zero showing the result

   Implementation:
   - Each step bar starts where the previous one ended
   - Negative components (taxes) are colored coral (#F26B44)
   - Positive components (government spending received) are colored sage (#7A9E7E)
   - The start bar (market income) is colored teal (#6EA1BE)
   - The final bar (net fiscal impact) is colored navy (#152F43)
   - Thin connector lines between bars showing the running total
   - Labels above or below each bar showing the dollar amount (formatted with $ and commas)

5. **NZ-born comparison**:
   - When enabled, render a second waterfall to the right with lighter/muted colors (same hue at 50% opacity)
   - Or render both as grouped bars at each step
   - Label which is which

6. **Summary stat**:
   - Below the chart: "Net fiscal contribution: -$15,400/year" in large text, colored by sign (sage if negative number means net contributor because taxes > spending, coral if net recipient)
   - Note: A negative "netFiscalImpact" means the person PAYS more than they receive (net contributor). Color this sage. A positive value means net recipient — color this coral.

7. **Loading/error states**: Skeleton with animate-pulse, error message

### Step 4: Apply design system

- Container: bg-[#F8F3EA] rounded-lg p-6
- Tax bars (subtracting): fill="#F26B44" (coral)
- Spending bars (adding): fill="#7A9E7E" (sage)
- Income bar: fill="#6EA1BE" (teal)
- Net impact bar: fill="#152F43" (navy)
- Connector lines: stroke="#D1D5DB" (light grey)
- Labels: fill="#152F43" (navy), font-size 11
- Axis: stroke="#6F899A" (steel)
