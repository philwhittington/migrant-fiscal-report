### Step 4: Define eligibility adjustments
Create adjustment functions:

```python
def adjust_nz_super(years_resident, age, base_nz_super):
    """NZ Super requires 10 years residence. Zero before that."""
    if years_resident < 10 or age < 65:
        return 0
    return base_nz_super

def adjust_wff(visa_category, base_wff):
    """WFF only for residents. Zero for temp workers, students, visitors."""
    if visa_category in ('Non-residential work', 'Student', 'Visitor', 'Australian'):
        return 0
    return base_wff

def adjust_working_age_support(years_resident, base_support):
    """50% discount for first 2 years (stand-down approximation)."""
    if years_resident < 2:
        return base_support * 0.5
    return base_support

def adjust_health(base_health):
    """85% of NZ average (healthy migrant effect)."""
    return base_health * 0.85

def adjust_education(base_education):
    """Apply at full NZ average rate for resident families."""
    return base_education
```

### Step 6: Compute NPV for each combination

Parameters:
- Discount rate: 0.035 (Treasury rate)
- Max projection: to age 85

For a migrant arriving at age A on visa V:
```python
npv = 0
for t in range(0, 86 - A):
    age = A + t
    age_band = get_5yr_band(age)  # e.g., 30 → "30-34"

    # Retention probability
    p_here = get_retention(V, t)

    # Base fiscal components from W&N
    base = fiscal_template[age_band]

    # Adjusted migrant fiscal impact
    migrant_tax = base['direct_taxes'] + tax_premium[V].get(nearest_10yr(age), 0)
    migrant_support = (
        adjust_nz_super(t, age, base['nz_super']) +
        adjust_wff(V, base['wff']) +
        adjust_working_age_support(t, base['working_age_support']) +
        base['housing_support'] * (0.5 if t < 2 else 1.0)
    )
    migrant_in_kind = (
        adjust_education(base['education_spending']) +
        adjust_health(base['health_spending'])
    )

    # Net fiscal impact for this year (negative = contributor)
    nfi = migrant_support + migrant_in_kind - migrant_tax

    # Discount and weight by retention
    discount = 1 / (1.035 ** t)
    npv += p_here * nfi * discount
```

Compute for:
- 7 visa categories × 8 arrival ages (20, 25, 30, 35, 40, 45, 50, 55)
- 11 nationalities × 4 arrival ages (25, 30, 40, 50)
- NZ-born baseline × 8 arrival ages (retention=1.0, no adjustments)
