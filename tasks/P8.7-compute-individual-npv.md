# Task P8.7: Compute individual NPV

## Objective
Compute the lifecycle net present value (NPV) for every synthetic individual by simulating from arrival to age 85. Each year's net fiscal impact is weighted by the probability of still being in NZ (retention curve), adjusted for age progression (income growth, changing expenditure profile), and discounted at 3.5%. This produces the distributional NPV data that powers the NPV distribution widget and policy scenario widget.

## Inputs
- `synth-pop/seed-population.parquet` (P8.6 output) — all columns including `net_fiscal_impact_annual`, `visa_category`, `age_start`, `years_since_residence`, `family_id`, all fiscal components
- `data/processed/retention-curves-by-visa.json` — retention probabilities by visa type and years since arrival
- `data/processed/wright-nguyen-fiscal-template.json` — for age-progression of expenditure components
- `synth-pop/utils.py` — import `compute_individual_npv`, `get_retention`, `get_wn_components`, `apply_migrant_adjustments`, `compute_paye`
- `synth-pop/config.py` — import `DISCOUNT_RATE` (0.035), `MAX_AGE` (85)
- `data/output/npv-by-visa-age.json` — Phase 1 reference for validation. Structure: array of objects with `visa`, `visa_code`, `arrival_age`, `npv`, `nzborn_npv`, `surplus`, `trajectory`

## Expected outputs
1. `synth-pop/seed-population.parquet` — **updated in place** with new columns:
   - `npv` (float): lifecycle NPV of this individual (present value of all future NFI, retention-weighted, discounted)
   - `npv_nzborn_equivalent` (float): NPV of a NZ-born person at the same arrival age (for comparison)
   - `surplus` (float): `npv - npv_nzborn_equivalent` (positive = migrant is better fiscal deal than NZ-born)

2. `data/output/synth-npv-distributions.json` — distributional summaries per `(visa_category, arrival_age)` cell:

### Step 2: Compute NPV for each individual
For each person, simulate from their current age to age 85. At each year:
1. Advance age by 1.
2. Advance `years_since_residence` by 1.
3. Look up retention probability for their visa type and years since arrival.
4. Recompute fiscal components for the new age (using W&N template + migrant adjustments).
5. Compute that year's NFI (revenue - expenditure).
6. Discount and weight by retention.

```python
# Map visa_category to a visa code that matches the retention data
# The retention data uses codes like 'Skilled Work', 'Family', etc.
# Build a mapping or use compute_individual_npv which handles this internally

seed['npv'] = 0.0
seed['npv_nzborn_equivalent'] = 0.0

# Vectorise by grouping: individuals in the same (visa_category, age_start, years_since_residence)
# group share the same lifecycle trajectory EXCEPT for income-dependent components.
# Strategy: compute a base NPV trajectory per group, then adjust for individual income.

# Simpler approach: use compute_individual_npv for each individual
# This is slow but correct. Optimise if needed.

for idx in range(len(seed)):
    row = seed.iloc[idx]
    npv = compute_individual_npv(
        arrival_age=row['age_start'],
        visa_code=row['visa_category'],
        gross_income=row['gross_income'],
        retention_data=retention_data,
        wn_template=wn_template,
        years_resident=row['years_since_residence'],
        relationship=row['relationship'],
        discount_rate=DISCOUNT_RATE,
        max_age=MAX_AGE,
    )
    seed.iat[idx, seed.columns.get_loc('npv')] = npv
```

**Performance**: 500k individual NPV calculations is expensive. Strategies:
1. **Group-level**: Compute NPV for each unique `(visa_category, age_start, income_decile, years_since_residence)` combination and broadcast.
2. **Vectorised**: Implement the NPV loop in numpy across all individuals simultaneously.
3. **Parallel**: Use `multiprocessing` or `joblib` to parallelise across groups.

Choose the approach that balances correctness with runtime. Target < 10 minutes.

### Step 3: Compute NZ-born equivalent NPV
For each individual, compute what a NZ-born person of the same age would contribute:
```python
for idx in range(len(seed)):
    row = seed.iloc[idx]
    nzborn_npv = compute_individual_npv(
        arrival_age=row['age_start'],
        visa_code='NZ-born',
        gross_income=row['gross_income'],  # same income
        retention_data=retention_data,
        wn_template=wn_template,
        years_resident=999,  # always resident
        relationship=row['relationship'],
        discount_rate=DISCOUNT_RATE,
        max_age=MAX_AGE,
    )
    seed.iat[idx, seed.columns.get_loc('npv_nzborn_equivalent')] = nzborn_npv

seed['surplus'] = seed['npv'] - seed['npv_nzborn_equivalent']
```

### Step 4: Build distributional summaries
```python
results = {}
for visa in seed['visa_category'].unique():
    for age in sorted(seed['age_start'].unique()):
        mask = (seed['visa_category'] == visa) & (seed['age_start'] == age)
        subset = seed.loc[mask, 'npv']
        if len(subset) < 10:
            continue

        key = f"{visa}|{age}"
        npv_vals = subset.values

        # Histogram with 20 bins
        hist_counts, bin_edges = np.histogram(npv_vals, bins=20)
        histogram = [
            {'bin_start': float(bin_edges[i]), 'bin_end': float(bin_edges[i+1]), 'count': int(hist_counts[i])}
            for i in range(len(hist_counts))
        ]

        results[key] = {
            'visa_category': visa,
            'arrival_age': int(age),
            'count': int(len(subset)),
            'mean_npv': float(np.mean(npv_vals)),
            'median_npv': float(np.median(npv_vals)),
            'p10_npv': float(np.percentile(npv_vals, 10)),
            'p25_npv': float(np.percentile(npv_vals, 25)),
            'p75_npv': float(np.percentile(npv_vals, 75)),
            'p90_npv': float(np.percentile(npv_vals, 90)),
            'std_npv': float(np.std(npv_vals)),
            'pct_net_contributor': float((npv_vals > 0).mean()),
            'histogram': histogram,
        }

with open('data/output/synth-npv-distributions.json', 'w') as f:
    json.dump(results, f, indent=2)
```

### Step 5: Build population summary
```python
summary = {
    'total_population': int(len(seed)),
    'total_npv': float(seed['npv'].sum()),
    'mean_npv': float(seed['npv'].mean()),

    'by_arrival_age': {},

for visa in seed['visa_category'].unique():
    subset = seed[seed['visa_category'] == visa]
    summary['by_visa_category'][visa] = {
        'count': int(len(subset)),
        'mean_npv': float(subset['npv'].mean()),
        'total_npv': float(subset['npv'].sum()),

for age in sorted(seed['age_start'].unique()):
    subset = seed[seed['age_start'] == age]
    summary['by_arrival_age'][str(age)] = {
        'count': int(len(subset)),
        'mean_npv': float(subset['npv'].mean()),

with open('data/output/synth-population-summary.json', 'w') as f:

```
