# Task P10.3: Household NPV widget

## Objective

Build a stacked bar chart widget showing the NPV contribution of complete family units — primary applicant, spouse, and children — rather than individuals alone. This addresses the policy-relevant question of whether a migrant family is a net fiscal contributor when all members are considered together, including dependants who consume public services but may not earn.

## Inputs

- `data/output/synth-household-npv.json` — family-unit NPV data by family configuration, visa type, and nationality
- `client/src/components/charts/NPVCalculatorWidget.tsx` — reference for widget pattern (useQuery, SVG, Tailwind, exports)
- `client/src/components/charts/chartRegistry.ts` — reference for lazy import pattern (do NOT modify; P10.5 handles registration)

## Expected outputs

- `client/src/components/charts/HouseholdNPVWidget.tsx` — a single self-contained React component

## Execution instructions

### Step 1: Read the existing widget pattern

Read `client/src/components/charts/NPVCalculatorWidget.tsx` for the standard pattern: useQuery, SVG charts, Tailwind, named export.

If the file does not exist, create a placeholder JSON with realistic values. Document assumptions.

### Step 3: Build the component

Create `client/src/components/charts/HouseholdNPVWidget.tsx` with:

1. **Imports**: React, useState, useQuery from @tanstack/react-query, cn from @/lib/utils

2. **Data fetching**: useQuery for `/data/synth-household-npv.json`

3. **Controls**:
   - Family configuration selector: `<select>` dropdown with all 4 configurations
   - Nationality selector: `<select>` dropdown
   - (Visa type is implicitly shown — all visa types displayed as bars)

4. **Stacked bar chart** (SVG, no D3):
   - One bar per visa type (X-axis: visa types)
   - Each bar is stacked by family member contribution:
     - Primary applicant: teal (#6EA1BE)
     - Spouse: sage (#7A9E7E)
     - Children: coral (#F26B44)
   - Y-axis: NPV in NZD
   - Zero line clearly marked
   - Child segments may extend below zero (negative NPV) — render them below the zero line
   - Each segment labeled with its dollar value on hover or as text annotation
   - Total family NPV shown as a label above/below each bar

5. **Key insight callout**: 
   - Prominent text box: "A skilled family of 4 still contributes $X net"
   - Dynamically updates based on selected configuration and nationality
   - Colored sage if positive, coral if negative

6. **Total family NPV**: Display prominently above the chart
   - Large text: "$XXX,XXX"
   - Label: "Total household net fiscal contribution"
   - Color: sage if positive, coral if negative

7. **Loading state**: Skeleton
8. **Error state**: Styled error message
