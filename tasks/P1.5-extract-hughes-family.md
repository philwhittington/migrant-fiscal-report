# Task P1.5: Extract Hughes Tables 9, 10, 11 (family composition and relationship tax)

## Objective
Extract tables showing how skilled migrants' tax payments and family composition vary by relationship to primary applicant (self, presumed spouse, presumed child), nationality, and years since arrival.

## Inputs
- `data/raw/hughes-an2602.xlsx` — Sheet "Data", rows where table_nbr in (9, 10, 11)

## Expected outputs
- `data/processed/hughes-table9-relationship-tax.json`
- `data/processed/hughes-table10-nationality-relationship.json`
- `data/processed/hughes-table11-tenure-tax.json`

## Execution instructions

Write a Python script `analysis/05-extract-hughes-family.py`:

### Table 9: Tax quantiles by relationship to primary applicant (~2,275 rows)
- var1=taxyr, var1_value=tax year (int)
- var2=age_10_bins, var2_value=age band start (int)
- var3=primary applicant relationship, var3_value=relationship (Self, Presumed Spouse, Presumed Child, Presumed Parent)
- var4=quantile, var4_value=quantile name (p10-p90)
- measure1=count (skip "S"), measure2=tax in dollars
- Output: `{taxyr, age_start, relationship, quantile, count, tax_dollars}`

### Table 10: Population by nationality and relationship (~1,205 rows)
- var1=taxyr, var1_value=tax year (int)
- var2=NULL
- var3=nationality_group, var3_value=nationality name
- var4=primary applicant relationship, var4_value=relationship
- measure1=count (skip "S"), measure2=NULL
- Output: `{taxyr, nationality, relationship, count}`
- Exclude rows where nationality is "NULL"
- Exclude "Presumed Parent" relationship (very small numbers)

### Table 11: Tax quantiles by relationship, age, and years since residence (~375 rows)
- var1=primary_app_reln, var1_value=relationship
- var2=age_10_bins, var2_value=age band start (int)
- var3=res_yrs_5bins, var3_value=years since first residence (0, 5, 10, 15, 20) (int)
- var4=quantile, var4_value=quantile name
- measure1=count (skip "S"), measure2=tax in dollars
- Output: `{relationship, age_start, residence_years_start, quantile, count, tax_dollars}`

Run the script.
