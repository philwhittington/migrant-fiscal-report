# Task P1.3: Extract Hughes Table 8 (tax by nationality/age/quantile)

## Objective
Extract Table 8 which contains tax payment distributions broken down by nationality group, age, and quantile. This is the key table for the nationality convergence analysis — showing how different nationality groups' tax payments have converged toward (or diverged from) NZ-born levels over 25 years.

## Inputs
- `data/raw/hughes-an2602.xlsx` — Sheet "Data", rows where table_nbr == 8

## Expected outputs
- `data/processed/hughes-table8-nationality.json`

## Execution instructions

Write a Python script `analysis/03-extract-hughes-nationality.py`:

1. Open the Excel file (read_only=True).
2. Filter to table_nbr == 8.
3. Column mapping:
   - var1=taxyr, var1_value=tax year (int)
   - var2=age_10_bins, var2_value=age band start (0,10,20,...) (int)
   - var3=nationality_group, var3_value=nationality name (string)
   - var4=quantile, var4_value=quantile name (p10, p25, p50, p75, p90)
   - measure1=underlying count (skip if "S"), measure2=tax in dollars at that quantile
4. Skip rows where measure1 == "S".
5. Output structure:
   ```json
   {
     "taxyr": 2024,
     "age_start": 30,
     "nationality": "South Asia",
     "quantile": "p50",
     "count": 51363,
     "tax_dollars": 14454.0
   }
   ```
6. Write to `data/processed/hughes-table8-nationality.json`.
7. Print summary: row count, distinct nationalities, year range.

Run the script.
