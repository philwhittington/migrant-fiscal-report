# Task P10.2: Fiscal waterfall with distributional bands widget

## Objective

Build a waterfall chart widget that extends the existing fiscal waterfall (Phase 1, point estimates) with distributional uncertainty bands. Each bar in the waterfall shows the p25-p75 range as a shaded band with whiskers extending to p10/p90, communicating that fiscal component estimates are not single numbers but distributions. This is a key visual for conveying uncertainty to policy audiences.

## Inputs

- `data/output/synth-fiscal-distributions.json` — distributional data for each fiscal component by visa type (percentiles for tax, GST, health, education, income support, NZ Super, net)
- `client/src/components/charts/FiscalWaterfallWidget.tsx` — the existing Phase 1 waterfall widget (reference for layout, color scheme, and waterfall bar logic)
- `client/src/components/charts/chartRegistry.ts` — reference for lazy import pattern (do NOT modify; P10.5 handles registration)

## Expected outputs

- `client/src/components/charts/FiscalWaterfallDistWidget.tsx` — a single self-contained React component

### Step 3: Build the component

Create `client/src/components/charts/FiscalWaterfallDistWidget.tsx` with:

1. **Imports**: React, useState, useQuery from @tanstack/react-query, cn from @/lib/utils

2. **Data fetching**: useQuery for `/data/synth-fiscal-distributions.json`

3. **Controls**:
   - Migrant type selector: `<select>` dropdown with all visa types

4. **Waterfall chart** (SVG, no D3):
   - Same waterfall layout logic as existing widget: bars flow left-to-right, each starting where the previous ended
   - Each bar shows the **mean** value as a horizontal line within the bar
   - The **p25-p75 range** is rendered as a semi-transparent shaded rectangle (same color as bar, 40% opacity)
   - **Whiskers** extend from p10 to p25 and from p75 to p90 as thin lines with caps
   - Bar widths are uniform; chart is horizontal (components on Y-axis, values on X-axis) or vertical (components on X-axis, values on Y-axis) — match the orientation of the existing waterfall
   - X-axis or Y-axis: NZD values with $ formatting
   - Component labels on the category axis

5. **Color scheme** (match existing waterfall):
   - Taxes (Income Tax, GST, ACC): coral (#F26B44)
   - Spending (Health, Education, Income Support, NZ Super): sage (#7A9E7E)
   - Income: teal (#6EA1BE)
   - Net: navy (#152F43)
   - Distributional bands: same color as parent bar at 40% opacity
   - Whisker lines: same color as parent bar at 70% opacity

6. **Legend**: Include legend items for "Mean", "p25-p75 range", "p10/p90 whiskers"

7. **Loading state**: Skeleton placeholder
8. **Error state**: Styled error message
