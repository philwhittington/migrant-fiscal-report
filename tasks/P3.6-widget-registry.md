## Expected outputs

- `client/src/components/charts/chartRegistry.ts` — updated with 5 new widget registrations
- `script/generate-static-data.ts` — updated to copy `data/output/*.json` files (this is the main deliverable for ensuring widgets have data at runtime; the full rewrite of this script happens in P4.1)

```typescript
// 3. Widget data files — copy all from data/output/
const outputDir = path.join(ROOT, "data", "output");
if (fs.existsSync(outputDir)) {
  const outputFiles = fs.readdirSync(outputDir).filter(f => f.endsWith(".json"));
  for (const file of outputFiles) {
    fs.copyFileSync(path.join(outputDir, file), path.join(PUBLIC_DATA_DIR, file));
    console.log(`  ✓ ${file} (from data/output/)`);
  }
}
```
