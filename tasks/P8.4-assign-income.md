# Task P8.4: Assign income

## Objective
Assign individual gross incomes to every person in the seed population by drawing from the fitted zero-inflated log-normal distributions (P8.2), then compute individual tax obligations (PAYE and ACC levy). This transforms the seed from a population skeleton into a population with income and tax data, ready for fiscal incidence analysis.

## Inputs
- `synth-pop/seed-population.parquet` (P8.1 output) — columns: `id`, `age_start`, `visa_subcategory`, `visa_category`
- `synth-pop/income-distributions.json` (P8.2 output) — fitted distribution params keyed by `"{visa_category}|{age_start}"`
- `synth-pop/utils.py` — import `sample_income`, `compute_paye`, `compute_acc_levy`
- `synth-pop/config.py` — import constants
- `data/processed/hughes-table4-visa-subcategory.json` — for per-capita mean tax validation

## Expected outputs
- `synth-pop/seed-population.parquet` — **updated in place** with new columns:
  - `gross_income` (float): annual gross income in dollars
  - `income_tax` (float): PAYE income tax in dollars
  - `acc_levy` (float): ACC earners' levy in dollars
  - `net_income` (float): gross_income - income_tax - acc_levy

## Execution instructions

### Step 1: Load inputs
```python
import json
import numpy as np
import pandas as pd
from synth_pop.utils import sample_income, compute_paye, compute_acc_levy
from synth_pop.config import TAX_YEAR

seed = pd.read_parquet('synth-pop/seed-population.parquet')

with open('synth-pop/income-distributions.json') as f:
    distributions = json.load(f)
```

### Step 2: Draw incomes per cell
Use a seeded RNG for reproducibility. For each `(visa_category, age_start)` group, look up the fitted distribution and draw incomes for all individuals in that group.

```python
rng = np.random.default_rng(seed=42)

seed['gross_income'] = 0.0

for key, params in distributions.items():
    visa_cat = params['visa_category']
    age = params['age_start']

    mask = (seed['visa_category'] == visa_cat) & (seed['age_start'] == age)
    n = mask.sum()

    if n == 0:
        continue

    incomes = sample_income(params, n, rng)
    seed.loc[mask, 'gross_income'] = incomes
```

### Step 3: Handle unmatched cells
Some seed individuals may not have a matching distribution (e.g. if Table 5 categories differ slightly from Table 4). For any unmatched individuals:
- Log a warning with the unmatched `(visa_category, age_start)` and count.
- Assign gross_income = 0 (conservative fallback).
- Write unmatched details to SCRATCHPAD.


### Step 4: Compute tax
```python
seed['income_tax'] = seed['gross_income'].apply(compute_paye)
seed['acc_levy'] = seed['gross_income'].apply(compute_acc_levy)
seed['net_income'] = seed['gross_income'] - seed['income_tax'] - seed['acc_levy']
```

### Step 5: Save
```python
seed.to_parquet('synth-pop/seed-population.parquet', index=False)
print(f"Updated seed: {len(seed):,} rows, {seed.columns.tolist()}")
print(f"Mean gross income: ${seed['gross_income'].mean():,.0f}")
```
