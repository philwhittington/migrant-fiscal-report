# Task P8.2: Fit income distributions

## Objective
Fit zero-inflated log-normal distributions to the quantile data in Hughes Table 5, calibrated against Table 4 per-capita mean tax. This produces a parameter file that P8.4 uses to draw individual incomes. The zero-inflation component captures the mass of individuals with zero or negligible tax (students, children, retirees), while the log-normal component captures the positive-income distribution.

## Inputs
- `data/processed/hughes-table5-visa-quantiles.json` — array of objects with fields:
  - `taxyr` (int): tax year
  - `age_start` (int): lower bound of 10-year age band (0, 10, 20, ...)
  - `visa_category` (str): e.g. "Resident", "Student", "Birth Citizen"
  - `quantile` (str): one of "p10", "p25", "p50", "p75", "p90"
  - `count` (int): cell population
  - `tax_dollars` (float): tax at this quantile in dollars
- `data/processed/hughes-table4-visa-subcategory.json` — for per-capita mean calculation:
  - `tax_billions` (float) and `count` (int) per `(visa_category, age_start)` cell
- `synth-pop/utils.py` — import `fit_zero_inflated_lognormal`
- `synth-pop/config.py` — import `TAX_YEAR`

## Expected outputs
- `synth-pop/income-distributions.json` — object keyed by `"{visa_category}|{age_start}"`, each value:
  ```json
  {
    "visa_category": "Resident",
    "age_start": 30,
    "p_zero": 0.15,
    "mu": 9.8,
    "sigma": 0.7,
    "fit_quality": 0.95,
    "calibration_mean": 12500,
    "source_quantiles": {"p10": 0, "p25": 3200, "p50": 8500, "p75": 15000, "p90": 25000},
    "count": 45000
  }
  ```

## Execution instructions

### Step 1: Load and filter Table 5 quantiles
```python
import json
import numpy as np
from synth_pop.config import TAX_YEAR
from synth_pop.utils import fit_zero_inflated_lognormal, sample_income

with open('data/processed/hughes-table5-visa-quantiles.json') as f:
    table5 = json.load(f)

# Filter to 2019
q2019 = [r for r in table5 if r['taxyr'] == TAX_YEAR]
```

### Step 2: Pivot quantiles into per-cell dicts
Group by `(visa_category, age_start)` and pivot the five quantile rows into a single dict:
```python
from collections import defaultdict
cells = defaultdict(dict)
for r in q2019:
    key = f"{r['visa_category']}|{r['age_start']}"
    cells[key][r['quantile']] = r['tax_dollars']
    cells[key]['count'] = r['count']
    cells[key]['visa_category'] = r['visa_category']
    cells[key]['age_start'] = r['age_start']
```

### Step 3: Compute per-capita mean tax from Table 4
```python
with open('data/processed/hughes-table4-visa-subcategory.json') as f:
    table4 = json.load(f)

# Aggregate Table 4 to visa_category level (sum across subcategories)
t4_means = {}
from collections import defaultdict
t4_agg = defaultdict(lambda: {'tax': 0.0, 'count': 0})
for r in table4:
    if r['year'] == TAX_YEAR and r['age_start'] is not None:
        key = f"{r['visa_category']}|{r['age_start']}"
        t4_agg[key]['tax'] += r['tax_billions'] * 1e9
        t4_agg[key]['count'] += r['count']

for key, v in t4_agg.items():
    t4_means[key] = v['tax'] / v['count'] if v['count'] > 0 else 0
```

### Step 4: Fit distributions
For each cell, call `fit_zero_inflated_lognormal()` with the five quantiles and the per-capita mean as calibration target:
```python
results = {}
for key, cell in cells.items():
    quantiles = {q: cell[q] for q in ['p10', 'p25', 'p50', 'p75', 'p90']}
    target_mean = t4_means.get(key, None)

    params = fit_zero_inflated_lognormal(quantiles, target_mean=target_mean)

    results[key] = {
        'visa_category': cell['visa_category'],
        'age_start': cell['age_start'],
        'p_zero': params['p_zero'],
        'mu': params['mu'],
        'sigma': params['sigma'],
        'fit_quality': params['fit_quality'],
        'calibration_mean': target_mean,
        'source_quantiles': quantiles,
        'count': cell['count'],
    }
```

### Step 5: Save
```python
with open('synth-pop/income-distributions.json', 'w') as f:
    json.dump(results, f, indent=2)

print(f"Fitted {len(results)} cells")
```
