# Task P8.3: Build assignment tables

## Objective
Build conditional probability tables for assigning nationality, relationship type, and years-since-residence to synthetic individuals. These tables encode the joint and conditional distributions from Hughes Tables 10 and 11, allowing P8.5 to draw attributes for each person.

## Inputs
- `data/processed/hughes-table10-nationality-relationship.json` — array of objects with fields:
  - `taxyr` (int): tax year
  - `nationality` (str): one of 11 nationality groups (e.g. "UK", "China", "South Africa", "Pacific", "South Asia", "Phillipines", "Other Asia", "Europe (excl UK)", "US/Canada", "Americas (excl US/Canada)", "Africa/Middle East (excl RSA)")
  - `relationship` (str): one of "Self", "Presumed Spouse", "Presumed Child"
  - `count` (int): number of people in this cell
- `data/processed/hughes-table11-tenure-tax.json` — array of objects with fields:
  - `relationship` (str): one of "Self", "Presumed Spouse", "Presumed Child"
  - `age_start` (int): lower bound of 10-year age band (0, 10, 20, ..., 70)
  - `residence_years_start` (int): lower bound of 5-year tenure band (0, 5, 10, 15, 20)
  - `quantile` (str): "p10", "p25", "p50", "p75", "p90"
  - `count` (int): number of people
  - `tax_dollars` (float): tax at this quantile
- `synth-pop/config.py` — import `TAX_YEAR`

## Expected outputs
- `synth-pop/assignment-tables.json` — object with three top-level keys:
  ```json
  {
    "nationality_marginal": {
      "UK": 0.25,
      "China": 0.12,
      ...
    },
    "relationship_given_nationality": {
      "UK": {"Self": 0.45, "Presumed Spouse": 0.30, "Presumed Child": 0.25},
      ...
    },
    "tenure_distribution": {
      "Self|20": {"0": 0.40, "5": 0.25, "10": 0.15, "15": 0.12, "20": 0.08},
      "Self|30": {...},
      "Presumed Spouse|30": {...},
      ...
    }
  }
  ```
  All probability vectors sum to 1.0 (within floating-point tolerance of 1e-9).

## Execution instructions

### Step 1: Load and filter Table 10
```python
import json
from collections import defaultdict
from synth_pop.config import TAX_YEAR

with open('data/processed/hughes-table10-nationality-relationship.json') as f:
    table10 = json.load(f)

t10_2019 = [r for r in table10 if r['taxyr'] == TAX_YEAR]
```

### Step 2: Compute nationality marginal distribution
```python
nat_counts = defaultdict(int)
for r in t10_2019:
    nat_counts[r['nationality']] += r['count']

total = sum(nat_counts.values())
nationality_marginal = {k: v / total for k, v in nat_counts.items()}
```

### Step 3: Compute P(relationship | nationality)
```python
nat_rel_counts = defaultdict(lambda: defaultdict(int))
for r in t10_2019:
    nat_rel_counts[r['nationality']][r['relationship']] += r['count']

relationship_given_nationality = {}
for nat, rels in nat_rel_counts.items():
    total_nat = sum(rels.values())
    relationship_given_nationality[nat] = {
        rel: count / total_nat for rel, count in rels.items()
    }
```

### Step 4: Build tenure distribution from Table 11
Table 11 has counts by (relationship, age_start, residence_years_start). Use the `count` field (not tax) to build P(tenure | relationship, age).

```python
with open('data/processed/hughes-table11-tenure-tax.json') as f:
    table11 = json.load(f)

# Deduplicate: Table 11 has multiple rows per (rel, age, tenure) — one per quantile.
# Take count from any quantile row (they should be the same).
tenure_cells = {}
for r in table11:
    key = (r['relationship'], r['age_start'], r['residence_years_start'])
    if key not in tenure_cells:
        tenure_cells[key] = r['count']

# Group by (relationship, age_start) and normalise
tenure_groups = defaultdict(lambda: defaultdict(int))
for (rel, age, tenure), count in tenure_cells.items():
    group_key = f"{rel}|{age}"
    tenure_groups[group_key][str(tenure)] = count

tenure_distribution = {}
for group_key, tenures in tenure_groups.items():
    total_group = sum(tenures.values())
    if total_group > 0:
        tenure_distribution[group_key] = {
            t: c / total_group for t, c in tenures.items()
        }
```

### Step 5: Assemble and save
```python
assignment_tables = {
    'nationality_marginal': nationality_marginal,
    'relationship_given_nationality': relationship_given_nationality,
    'tenure_distribution': tenure_distribution,
}

with open('synth-pop/assignment-tables.json', 'w') as f:
    json.dump(assignment_tables, f, indent=2)

print(f"Nationalities: {len(nationality_marginal)}")
print(f"Tenure groups: {len(tenure_distribution)}")
```
