# Task P3.3: Retention curve explorer widget

## Objective

Build a Retention Curve Explorer widget that shows what proportion of migrants in each visa category remain resident in New Zealand over time. Retention is a key driver of fiscal impact — a migrant who leaves after 3 years contributes far less than one who stays 15+. This widget makes that story interactive.

## Inputs

- `data/processed/retention-curves-by-visa.json` — retention percentages by visa type and years since arrival
- `client/src/components/charts/VATDemoWidget.tsx` — reference for existing widget pattern
- `client/src/components/charts/chartRegistry.ts` — reference for registration (do NOT modify; P3.6 handles this)

## Expected outputs

- `client/src/components/charts/RetentionExplorerWidget.tsx` — the React widget component

## Execution instructions

If the file does not exist, create a placeholder `data/processed/retention-curves-by-visa.json` with realistic values. Use 19 data points (years 0-18 since arrival). Skilled and Family visa holders have high retention (~65-70% at year 18). Students and Temp Work have low retention (~14-18% at year 18). Humanitarian falls in between.

### Step 2: Read the existing widget pattern

Read `client/src/components/charts/VATDemoWidget.tsx` to understand data fetching (useQuery, fetch URL pattern), loading/error handling, Tailwind styling, and export conventions.

### Step 3: Build the component

Create `client/src/components/charts/RetentionExplorerWidget.tsx` with:

1. **Imports**: React, useState, useQuery from @tanstack/react-query, cn from @/lib/utils

2. **Data fetching**: useQuery to fetch `/data/retention-curves-by-visa.json`

3. **Controls**:
   - Multi-select checkboxes, one per visa type. Default checked: Skilled, Family, Temp Work, Student.
   - Each checkbox label colored with its visa type color.
   - Layout: horizontal row of checkboxes, wrapping on mobile.

4. **Visa type colors** (assign consistently):
   - Skilled: #6EA1BE (teal)
   - Family: #7A9E7E (sage)
   - Humanitarian: #C9A227 (gold)
   - Australian: #F26B44 (coral)
   - Temp Work: #163E5F (deep blue)
   - Student: #6F899A (steel)

5. **Line chart** (plain SVG, no external library):
   - X-axis: years since arrival (0-18), labelled at 0, 3, 6, 9, 12, 15, 18
   - Y-axis: percentage still resident (0-100%), labelled at 0, 25, 50, 75, 100
   - One line per selected visa type in its assigned color, line width 2
   - Shaded area under each curve (same color, opacity 0.1)
   - Grid lines at major Y ticks (light grey, dashed)
   - SVG viewBox for responsiveness

6. **Annotations**:
   - For Student visa: if visible, annotate the point where retention drops below 50% with a small marker and text "50% of students leave within ~4 years" (adjust the year to match the actual data)
   - Position annotations to avoid overlapping lines

7. **Cohort size display**:
   - Below the chart, show a row of small stat cards for each visible visa type: "Skilled: 45,000 initial cohort"
   - Styled with the visa type color as left border

8. **Loading/error states**: Skeleton with animate-pulse, styled error message

### Step 4: Apply design system

- Widget container: bg-[#F8F3EA] rounded-lg p-6
- Heading: text-[#152F43] font-semibold
- Chart background: white or very light
- Grid lines: stroke="#E5E7EB" (light grey)
- Axis labels: fill="#6F899A" (steel), font-size 12
- Annotation text: fill="#152F43", font-size 11

### Step 5: Export correctly

```typescript
export function RetentionExplorerWidget() { ... }
export default RetentionExplorerWidget;
```

## Self-check
