# Task P8.1: Build seed population

## Objective
Build a synthetic seed population of ~500,000 individuals from Hughes Table 4 (visa subcategory by age band), filtered to tax year 2019. Each row in the output represents one synthetic person, stratified by visa subcategory and age band. This seed becomes the skeleton that downstream tasks attach income, attributes, and fiscal calculations to.

## Inputs
- `data/processed/hughes-table4-visa-subcategory.json` — array of objects with fields:
  - `year` (int): tax year
  - `age_start` (int or null): lower bound of 10-year age band (0, 10, 20, ..., 100; null = aggregate)
  - `visa_subcategory` (str): e.g. "R.Family", "S.Scholarship", "C.Birth_citizen"
  - `visa_category` (str): e.g. "Resident", "Student", "Birth Citizen"
  - `count` (int): number of people in this cell
  - `tax_billions` (float): total tax paid by this cell in $B
- `synth-pop/config.py` — import `POPULATION_TARGET` (500,000) and `TAX_YEAR` (2019)

## Expected outputs
- `synth-pop/seed-population.parquet` — Parquet file with columns:
  - `id` (int): unique identifier 0..N-1
  - `age_start` (int): lower bound of 10-year age band (0, 10, 20, ..., 100)
  - `visa_subcategory` (str): original subcategory code
  - `visa_category` (str): parent category
- Row count: approximately 500,000 (within 1% tolerance)

## Execution instructions

### Step 1: Read and filter Table 4
```python
import json
import numpy as np
import pandas as pd
from synth_pop.config import POPULATION_TARGET, TAX_YEAR

with open('data/processed/hughes-table4-visa-subcategory.json') as f:
    table4 = json.load(f)

# Filter to 2019, exclude null age_start (aggregate rows) and zero-count cells
cells = [
    r for r in table4
    if r['year'] == TAX_YEAR
    and r['age_start'] is not None
    and r['count'] > 0
]
```

### Step 2: Compute sampling weights
The total real population in 2019 with age bands is ~4.7M. We need a stratified subsample of ~500k that preserves the proportional composition.

```python
total_population = sum(c['count'] for c in cells)
sampling_fraction = POPULATION_TARGET / total_population  # ~0.105

# Compute target count per cell (round, ensure at least 1 per non-empty cell)
for c in cells:
    c['target_count'] = max(1, round(c['count'] * sampling_fraction))

actual_target = sum(c['target_count'] for c in cells)
```

### Step 3: Expand to individual rows
```python
rows = []
person_id = 0
for c in cells:
    for _ in range(c['target_count']):
        rows.append({
            'id': person_id,
            'age_start': c['age_start'],
            'visa_subcategory': c['visa_subcategory'],
            'visa_category': c['visa_category'],
        })
        person_id += 1

seed = pd.DataFrame(rows)
```

### Step 4: Save as Parquet
```python
seed.to_parquet('synth-pop/seed-population.parquet', index=False)
print(f"Seed population: {len(seed):,} rows")
```
