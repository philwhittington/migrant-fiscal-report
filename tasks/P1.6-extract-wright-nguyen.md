# Task P1.6: Extract Wright and Nguyen fiscal incidence template

## Objective
Parse the Wright & Nguyen AN 24/09 CSV file into structured JSON. This dataset provides the expenditure side of the fiscal equation — how much government spends on education, health, NZ Super, benefits, and housing for people at each age. Combined with Hughes's migrant tax data, it enables the net fiscal impact calculation.

## Inputs
- `data/raw/wright-nguyen-an2409.csv`

## Expected outputs
- `data/processed/wright-nguyen-fiscal-template.json`

## Execution instructions

Write a Python script `analysis/06-extract-wright-nguyen.py`:

The CSV has columns: Statistic, Sharing assumption, Unit, Group or category, Sample base, Population base, Population base sampling error, Estimate, Sampling error, Note

Extract three datasets:

### 1. Fiscal components by age (No sharing assumption)

For each age group matching pattern "Age X-Y" (17 groups: Age 0-4 through Age 80+), extract all dollar-value rows under "No sharing" sharing assumption:

Key statistics to extract:
- Average Market income
- Average Direct taxes
- Average Income support
- Average Working for Families
- Average NZ Super and Vets
- Average Working-age support
- Average Housing support
- Average Paid Parental Leave
- Average Student Allowance
- Average Education spending
- Average Health spending
- Average Disposable income
- Average Independent Earner Tax Credit
- Average Winter Energy Payment
- Average Other Non-Taxable Benefits
- Average Other income support

Output structure:
```json
{
  "age_band": "30-34",
  "components": {
    "market_income": 55090,
    "direct_taxes": 11360,
    "income_support": 4410,
    "wff": 1390,
    "nz_super": 0,
    "working_age_support": 1430,
    "housing_support": 830,
    "paid_parental_leave": 390,
    "education_spending": 970,
    "health_spending": 2650,
    "disposable_income": 48150,
    "other_income_support": 770
  }
}
```

Skip rows where Estimate is empty or non-numeric.

### 2. Net fiscal impact by age (Family sharing)

For groups matching "Principal Age X-Y", extract under "Family" sharing:
- Average Net Fiscal Impact
- Average Final income
- Average Market income
- Average Direct taxes
- Average Income support
- Average Indirect taxes
- Average In-kind spending

Output structure:
```json
{
  "age_band": "30-34",
  "net_fiscal_impact": -3830,
  "nfi_error": 2770,
  "final_income": 52340,
  "market_income": 53200,
  "direct_taxes": 16970,
  "income_support": 6890,
  "indirect_taxes": 7110,
  "in_kind_spending": 13360
}
```

### 3. Family composition by age

For "Principal Age X-Y" groups, extract:
- Couple families (unit=Families)
- Couple-with-children families (unit=Families)
- Single-member families (unit=Families)
- Sole-parent families (unit=Families)

Output structure:
```json
{
  "age_band": "30-34",
  "couple": 46000,
  "couple_with_children": 80000,
  "single_member": 79000,
  "sole_parent": 17000
}
```

Combine all three into a single JSON with keys: `fiscal_components`, `net_fiscal_impact`, `family_composition`.

Run the script.
